👤 Borrowers API

POST /api/borrowers

Create new driver borrower profile. 


{
"name":"John Doe",
"phone":"+250700000000",
"permit":"A123456",
"location":"Kigali"
}
GET /api/borrowers

Retrieve all borrowers.

DELETE /api/borrowers/{id}

Remove borrower record.

POST /api/assets

Register vehicle asset. 


{
"type":"Truck",
"plate":"RAB123A",

}
GET /api/assets

💳 Loans API

POST /api/loans

Create financing loan. 


{
"borrower_id":1,
"principal":5000000,
"period_months":36,
"weekly_payment":50000
}
GET /api/loans/{id}

Retrieve loan details.

💰 Payments API

POST /api/payments

Record repayment. 


{
"loan_id":1,
"amount":50000,
"method":"Mobile Money",
"status":"Paid"
}
GET /api/payments/history/{loan_id}

Get payment history.

📊 Reports API

GET /api/reports/portfolio

Get total loan portfolio statistics.

GET /api/reports/collections

Get payment collection reports.
